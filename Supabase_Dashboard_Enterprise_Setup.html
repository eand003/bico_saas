Administração SaaS

Painel Supabase: Configuração de Parceiros

Passo 01

Acessar o Gerenciamento de Usuários

Faça login no Dashboard do Supabase, selecione o seu projeto e vá em Authentication > Users na barra lateral esquerda.

Passo 02

Selecionar o Usuário da Revenda

Selecione o e-mail que pertence à loja (ou crie um novo para o cliente). Clique nos três pontinhos (Vertical Dots) à direita e escolha Edit User Metadata.

Passo 03

Injetar os Dados de Branding

No editor de JSON que abrir, vá no campo User Metadata e adicione as chaves abaixo (substituindo pelos dados da loja do seu cliente):

{
  "partner_name": "Agro Peças John Deere",
  "partner_color": "#367c2b",
  "partner_logo": "https://imgur.com/link-da-logo-jd.png",
  "partner_whatsapp": "5565988887777"
}

IMPORTANTE: O partner_color deve ser um código HEX (Hexadecimal) para que o sistema altere todas as cores do app instantaneamente.

Mapeamento de Efeito Visual

Campo JSON O que ele muda no App?
partner_name Título do Cabeçalho e Título da Página (SEO).
partner_color Botões, Ícones, Gráficos e o Hover dos cards.
partner_logo Imagem no topo e na tela de login.
partner_whatsapp Número para onde os botões de "Comprar" enviam o cliente.
Dica do Desenvolvedor: Ao trocar a logo, certifique-se de que a imagem tenha o fundo transparente (PNG) ou uma cor que combine com o Dark Mode para um resultado profissional.
